# Fucking

一个调用网页 *API* 进行骂人Python自动骂人工具!!!



**运行前请安装依赖库(pip install +库名)**

- PyQt5
- requests
- pynput
- pyperclip
- lxml



双击 **运行.pyw **运行，开启后点击要发送的文本框，自动发送（发送按键设置为 **回车键** ）

功能: 可选择夸人、骂人，句子可从文本文档导入，也可直接用骂人网站里的句子。
